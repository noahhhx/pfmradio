# PFM Radio

You are a very experienced developer tasked with making a website to play msuic in a Radio style.
This is currently an empty project - following are the instructions and requirements.

- [x] Use Vue front-end framework with typescript
- [x] Use Tailwind CSS
- [x] There should only be a single home page
- [x] There are no users or authentication
- [x] We will only be playing music from videos that are on youtube
- [x] There is no need for a database
- [x] The page should have a currently playing section
    - [x] Includes thumbnail of the video
    - [x] Has skip functionality
    - [x] Shows up next
- [x] You should be able to queue a YouTube video to be played next
- [x] There should be a 'Mix Playlist' where users can submit a mix too
- [x] The 'Mix Playlist' is what plays be default if nothing is queued
- [x] Users should be able to submit YouTube playlists, to add to the list of playlists
- [x] Create a dockerfile for deployment

Remember simplicity is key - this is intended for personal (and friends use).

Update the README as you go.
Any problems created a new markdown file with TODOs.
As you complete tasks, check them off on this file.
